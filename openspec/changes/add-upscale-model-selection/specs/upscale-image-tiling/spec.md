## ADDED Requirements

### Requirement: 固定形状タイルへの分割と外周パディング

システムは、任意寸法の入力画像を、ONNX モデルが要求する**固定タイルサイズ**の等寸タイル列に分割しなければならない (MUST)。入力寸法がタイルサイズの整数倍でない場合、右端・下端のタイルが欠けないよう外周を uniform にパディングして全タイルを同一寸法に揃えなければならない (MUST)。これは NNAPI が動的入力形状を非対応であり、CoreML が固定形状で高性能であるという制約に対応する。

#### Scenario: 非整数倍の入力を等寸タイルに分割する

- **WHEN** 入力画像の幅または高さが固定タイルサイズの整数倍でない
- **THEN** 外周パディングにより全タイルが固定タイルサイズに揃い、欠けるタイルは生じない

#### Scenario: 整数倍の入力は外周パディング不要

- **WHEN** 入力画像の幅と高さが固定タイルサイズの整数倍である
- **THEN** 余分な外周パディングを加えず、タイル列はちょうど入力を覆う

### Requirement: タイル毎内側パディングによるエッジ artifact 抑制

システムは、各タイルの推論前に、タイル境界の畳み込みエッジ artifact を抑制するための**内側パディング（既定 ~10px）**を隣接画素から付与しなければならない (MUST)。推論後、内側パディング相当の領域は scale 倍した上でクロップ除去し、隣接タイルと継ぎ目なく再合成できなければならない (MUST)。

#### Scenario: 内側パディングが推論後にクロップされる

- **GIVEN** 内側パディング px を持つタイルを scale 倍で推論する
- **WHEN** タイル出力を再合成する
- **THEN** 内側パディング相当（scale 倍）の領域はクロップされ、最終画像に境界の継ぎ目が現れない

### Requirement: scale 依存の入力可分性（divisibility）保証

システムは、モデルへ渡すタイル入力が、scale ごとに要求される可分性を満たすことを保証しなければならない (MUST)。scale=2 では入力辺長が 2 の倍数でなければならず、scale=4 では追加の可分性制約を課さない。固定タイル経路ではタイル辺長を当該倍数で割り切れる値に選ぶことでこれを満たす（その場合 reflect-padding は不要）。動的入力（全画像）経路を採る場合は、入力辺長を当該倍数へ reflect-padding で切り上げ、出力で相当領域を scale 倍してクロップ除去しなければならない (MUST)。

#### Scenario: scale=2 のタイル辺長は 2 の倍数である

- **WHEN** scale=2 の固定タイル経路を構成する
- **THEN** タイル辺長は 2 の倍数で、モデル入力は追加 padding なしに可分性を満たす

#### Scenario: scale=4 は追加の可分性制約を課さない

- **WHEN** scale=4 のタイル経路で推論する
- **THEN** 追加の mod-padding は不要で、出力寸法は valid 核領域のちょうど 4 倍である

### Requirement: 出力寸法と継ぎ目なし再合成

再合成後の最終画像は、外周パディング・内側パディング・mod-padding をすべてクロップ除去した上で、元入力寸法のちょうど scale 倍でなければならない (MUST)。

#### Scenario: 最終寸法が scale 倍になる

- **GIVEN** 寸法 W×H の入力と scale 係数 s
- **WHEN** タイル分割・推論・再合成を行う
- **THEN** 最終画像の寸法は (W·s)×(H·s) で、全パディングはクロップ済みである

### Requirement: タイリング前処理は決定的で純粋にテストできる

タイル分割・パディング・クロップ・再合成のロジックは、実モデル・GPU・ネットワークなしで決定的に検証できなければならない (MUST)。`flutter test` でローカル・全 CI ジョブにおいて通過しなければならない (MUST)。

#### Scenario: 既知入力で分割と再合成が往復一致する

- **GIVEN** 既知寸法の入力と固定タイルサイズ
- **WHEN** 分割 → （恒等な擬似推論）→ 再合成を行う
- **THEN** タイル数・各タイル寸法・最終寸法が期待どおりで、実モデルやネットワークを必要としない
